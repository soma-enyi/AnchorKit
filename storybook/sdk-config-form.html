SDK Config Form

A form component for configuring AnchorKit SDK settings including network selection, anchor domain, timeout, and custom HTTP headers.

Interactive Demo

üìã Component Props

Prop Type Required Default Description
network 'testnet' | 'mainnet' Yes 'testnet' Stellar network to connect to
anchorDomain string Yes - Domain of the anchor service (3-253 characters)
timeoutSeconds number No 30 Request timeout in seconds (1-300)
customHeaders Array<{key, value}> No [] Custom HTTP headers to include in requests
onSubmit Function No - Callback when form is submitted with valid config
onError Function No - Callback when validation fails

üíª Usage Example

// Rust usage - creating SDK config
use anchor_kit::config::SdkConfig;

let config = SdkConfig::builder()
  .network(SdkNetwork::Testnet)
  .anchor_domain("demo.anchor.com")
  .timeout_seconds(30)
  .custom_header("Authorization", "Bearer token")
  .build()
  .expect("Failed to build config");
