Status Monitor

A component for monitoring the health status of Anchor service endpoints. Displays online, degraded, and offline states with real-time status updates.

Interactive Demo

üìã Component Props

Prop Type Required Default Description
baseUrl string Yes - Base URL of the anchor service
endpoints Endpoint[] No [/info, /auth, /transactions] List of endpoints to monitor
refreshInterval number No 30000 Auto-refresh interval in milliseconds
timeout number No 5000 Request timeout in milliseconds
onStatusChange Function No - Callback when endpoint status changes

üíª Usage Example

// Rust usage - Status monitoring

// Check single endpoint

// Check all endpoints
